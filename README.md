# GameHunt
# [Powered by RAWG](https://rawg.io/)


  A React-Native mobile app that displays Game Information
  
  To Install
  
  Make sure you have expo-cli installed
  
  Clone or Download the repository
  
  Go to the proj directory and install using:
  
  ```npm install```
  
  Start using:
  
  ```expo start```
